Deploying the Minecraft server website to an Ubuntu VPS involves these steps:

1. **Prepare the server** - Install Node.js, npm, and set up environment
2. **Clone and build** - Transfer code, install dependencies, build for production
3. **Configure environment** - Set up .env with production values
4. **Set up PM2** - Process manager for keeping Next.js running
5. **Deploy and monitor** - Start the app, configure HTTPS with Let's Encrypt

See `DEPLOYMENT.md` for full instructions. [[ubuntu-deployment-guide]]
